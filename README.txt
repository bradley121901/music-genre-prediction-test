Made in Jupyter with Pandas and Scikit-learn Python libraries for making music genre predictions from an arbitrary dataset using a decision tree.
This respiratory contains 5 files, this README.txt file, the music.csv dataset file, MusicGenrePredicationTest.ipynb which contains the main code, 
music-recommender.dot contains a visualization of the decision tree alorgithmn, and a music-recommender.joblib file for persisting the model.
